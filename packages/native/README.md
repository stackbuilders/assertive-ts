# Assertive.ts Native

An Assertive.ts plugin that provides assertion Matchers for React Native elements.

## Requirements

- **@assertive-ts/core:** >=2.0.0

## Install

```sh
npm install --save-dev @assertive-ts/native
```

Or:

```sh
yarn add --dev @assertive-ts/native
```
